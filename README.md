# Mass_Printing
PowerShell-скрипт для массовой печати документов в Windows.

## Требования:
• Windows 7-11  
• Принтер  

## Как использовать:
1. Дважды кликните по файлу `Start_Mass_Printing.bat`  
2. Откройте папку с файлами для печати  
3. Выберите файлы для печати (для выбора всех файлов в папке используйте комбинацию Ctrl + A)  
4. Выберите количество копий для печати  
5. Дождитесь окончания печати

**Обратите внимание:**  
Скрипт сортирует файлы для печати в алфавитном и числовом порядке.  
Опционально после печати распечатанные файлы перемещаются в папку `Распечатано_Текущая дата`.
